医療用語辞書
Android・Gboard向け

LINEやメールで看護師・スタッフへ指示を送るとき、医療用語や薬剤名がすぐ変換候補に出るようにするための辞書導入アプリです。

匿名アンケートに答える

APK本体が配布フォルダへ配置されると、ダウンロードボタンが自動で有効になります。

42,436収録エントリ
無料広告・課金なし
Android 10+対象OS
Gboard対応キーボード

できること

  • 医療用語、病名、薬剤名、検査名、医療略語などをGboardの変換候補へ追加します。
  • 登録後はLINE、Gmail、メモなど、Gboardを使う各アプリで利用できます。
  • アプリが辞書ZIPを端末へ保存し、Gboardへのインポート手順を案内します。
  • 通信、広告、アクセス解析、アカウント登録はありません。
実機テストでは、通常の辞書では出にくい医療用語が変換候補の上位に表示されることを確認しています。

使い方

1

アプリをインストール

このページからAPKをダウンロードし、Android端末へインストールします。

2

辞書ZIPを保存

アプリ内の「辞書ファイルを保存」を押し、ダウンロードフォルダへ保存します。

3

Gboardへインポート

画面キーボード設定からGboardの単語リストを開き、保存したZIPをインポートします。

Gboard側の操作

  1. Gboard設定を開く
  2. 「単語リスト」→「単語リスト」→「日本語」を選ぶ
  3. 右上のメニューから「インポート」を選ぶ
  4. medical_dict_gboard_v2026.07.05.zipを選ぶ

利用前にご確認ください

  • アプリの識別子(applicationId)を変更したため、以前の版がインストールされている場合は上書き更新できません。 一度以前のアプリをアンインストールしてから、新しいアプリをインストールしてください。
  • すでにGboardへ辞書を登録済みの場合、アプリを入れ直しても再インポートは不要です。 Gboardの単語リストはアプリとは別に保持されます。同じ辞書を重ねてインポートすると重複登録になるためご注意ください。
  • Gboard専用です。Samsungキーボード、ATOK、Simeji等では利用できません。
  • 既存のユーザー辞書がある場合は、先にGboardからバックアップしてください。
  • 端末やGboardの状態により、全42,436語が登録されない場合があります（登録されない語が一定数生じることを実機で確認しています）。
  • アプリを削除しても、Gboardへ登録済みの単語は自動では削除されません。
  • 辞書更新時は再インポートが必要で、旧版との重複が生じる可能性があります。
  • 医療行為の判断支援をするアプリではありません。変換候補の内容は送信前に必ず確認してください。
Windows向け

WindowsのMicrosoft IMEでも使えます

Windowsをお使いの方は、標準のMicrosoft IMEに取り込めます。取り込むと、Word・メール・ブラウザなどWindows全体の文字入力で医療用語が変換候補に出ます（Gboardのようにアプリは不要です）。

MS-IME用辞書ファイルをダウンロード

取り込み手順（Windows）

  1. 上のファイル（medical_dict_msime_v2026.7.5.txt）をダウンロードします。
  2. 画面右下の入力モード表示「あ / A」を右クリックし、「単語の追加」を開きます。
  3. 開いた画面の「ツール」メニュー →「テキスト ファイルからの登録」を選びます。
  4. ダウンロードしたファイルを選ぶと、辞書に一括登録されます（数万語のため少し時間がかかります）。
  5. 「よみ」を入力すると、医療用語が変換候補に表示されます。
Windows 11のMicrosoft IMEで、医療用語（哆開・痘瘡・家族性良性慢性天疱瘡など、通常は変換できない語）が候補に出ることを確認済みです。
  • 初めて使う長い用語は、読みを最後まで入力すると候補に出ます。一度使うとMS-IMEが学習し、次からは数文字で出るようになります（よく使う用語ほど速くなります）。
  • 「単語の追加」の画面が出ない場合は、Microsoft IMEの設定で「以前のバージョンの Microsoft IME を使う」をオンにすると確実です。
Mac / iPhone向け（試験版）

Mac・iPhoneでも使いたい方へ

AndroidのGboardとは別に、macOSの「ユーザ辞書」に取り込む形式も用意しました。Macで取り込むと、iCloud経由でiPhone・iPadにも同期され、標準キーボードで医療用語が変換候補に出ます。App Storeアプリではなく辞書ファイルを取り込む方式のため、iPhone単体では取り込めませんが、Macをお持ちの方はiPhoneでも利用できます。

ユーザ辞書ファイル（.plist）をダウンロード

取り込み手順（Mac）

  1. Macで上のファイル（medical_dict_apple_v2026.7.5.plist）をダウンロードします。
  2. 「システム設定」→「キーボード」→「テキスト変換（ユーザ辞書）」を開きます。
  3. ダウンロードした .plist を一覧へドラッグ＆ドロップします。
  4. iCloud（同じApple ID）がオンなら、iPhone・iPadの「設定 → 一般 → キーボード → ユーザ辞書」へ自動同期されます。
  5. 「よみ」を入力すると、医療用語が変換候補に表示されます。
試験版です（動作は未検証・フィードバック募集）。
  • 開発側にmac環境がなく、実機での動作確認ができていません。ご利用の結果を メール でお知らせいただけると助かります。
  • 取り込みにはMacが必要です。iPhone単体では一括取り込みができません。
  • iOSのユーザ辞書は登録数が多いと同期が不安定になる場合があり、約42,000語すべてが同期・表示されるとは限りません。
  • ユーザ辞書は「よみ」で引く方式のため、同じ読みに複数の医療用語がある場合、一部しか表示されないことがあります。
  • 読みは（ほぼ）最後まで入力する必要があります。一度使った語は学習され、次回から出やすくなります。
